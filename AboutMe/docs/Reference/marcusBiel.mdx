# Marcus Biel

   Distinguished Software Architect and Developer

Ich habe fast zwei Jahre mit Milancho an einem großen Projekt zusammengearbeitet. Gemeinsam haben wir ein Java-Backend-System von Java 8 auf Java 22 migriert, unter Verwendung von Spring Boot 3.3 und einer hexagonalen Architektur auf Basis von DDD. Dadurch wurde das System modular, wartbar und für die 30 Entwickler im Team flexibel erweiterbar. Zusätzlich haben wir Integrationstests mit RestAssured und JUnit 5 eingeführt. Milancho ist ein erfahrener Architekt und Senior Entwickler mit fundiertem Wissen in Clean Code, Clean Architecture und DDD. Die Zusammenarbeit mit ihm macht großen Spaß – er ist ein hervorragender Teamplayer und ein toller Mensch. Ich empfehle ihn uneingeschränkt für jede Art von komplexen Java-/Cloud-Projekten
